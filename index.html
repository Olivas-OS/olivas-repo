🫒 Olivas OS APT Repository

Resmi Olivas OS paket deposu


Sisteminize eklemek için:

curl -fsSL https://Olivas-OS.github.io/olivas-repo/olivas.gpg | sudo gpg --dearmor -o /etc/apt/trusted.gpg.d/olivas.gpg echo "deb https://Olivas-OS.github.io/olivas-repo stable main" | sudo tee /etc/apt/sources.list.d/olivas.list sudo apt update && sudo apt install olivas-store
